# arm-vibe (vibeOS)

Модульная ARM64 операционная система с графическим интерфейсом, поддержкой двух режимов (desktop и phone) и установкой через ISO.

## Цели проекта
- Модульная архитектура для легкого расширения.
- Поддержка ARM64 архитектуры.
- Запуск в QEMU для тестирования.
- Графический интерфейс с адаптацией под desktop и phone режимы.
- Установка через ISO образ.

## Структура проекта
- `kernel/` - Ядро ОС.
- `bootloader/` - Загрузчик (EFI-based для ARM64).
- `gui/` - Графический интерфейс.
- `drivers/` - Драйверы устройств.
- `libs/` - Общие библиотеки.
- `tests/` - Тесты.
- `docs/` - Документация.

## Сборка и запуск
1. `make build` - Собрать ОС.
2. `make run` - Запустить в QEMU ARM64 с VNC на localhost:5900.
   - Подключитесь VNC viewer (например, `vncviewer localhost:5900`) для просмотра видео.
   - Serial output в терминале.
